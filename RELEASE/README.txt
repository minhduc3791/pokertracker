PokerTracker v1.0
==================

A real-time poker statistics HUD overlay for Natural8.

System Requirements
-------------------
- Windows 10 or Windows 11
- Natural8 poker client installed
- Administrator privileges (for process detection)

Features
--------
- Real-time HUD overlay showing player statistics
- Tracks VPIP, 3Bet, WTSD, W$SD and more
- Automatic hand history parsing
- Multi-table support
- System tray integration
- Background operation

Installation
------------
1. Extract PokerTracker.exe from this folder
2. Run PokerTracker.exe (no installation required)

First-Time Setup
---------------
1. Launch PokerTracker
2. Configure your hero screen name in Settings
3. Point to your Natural8 hand history folder:
   C:\Users\<YourName>\AppData\Local\Gognac\NG\HandHistory

Usage
-----
1. Launch PokerTracker before starting Natural8
2. The HUD will appear when Natural8 tables are open
3. Right-click the system tray icon to:
   - Show/Hide HUD
   - Open Settings
   - Quit

Statistics Displayed
--------------------
- VPIP: Voluntarily Put In Pot (pre-flop call/raise)
- 3B: 3-Bet percentage
- WTSD: Went to Showdown
- W$SD: Won Money at Showdown

Notes
-----
- The HUD appears on the right side of your screen
- Player stats update in real-time as hands are played
- Stats are stored locally in poker_tracker.db
